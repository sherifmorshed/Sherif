📋 How It Works

Upload your Excel file (LAND_WELLS_SITUATION.xlsx) and the app will read it and update all well data instantly — no internet needed, no re-deployment.

  • Sheet used: DATA BASE
  • Header row: Row 2 (No., Well, Type, Status...)
  • All existing test logs and W.C overrides are kept
  • Data is saved in the browser for next time
